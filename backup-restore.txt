dotfiles all go on git in a .dotfiles folder.

Restore procedure:
$ apt-get install git # Or manually
$ git clone https://github.com/andrewg/.dotfiles .dotfiles
$ cd .dotfiles
$ ./install.sh

install.sh:
-----------
- Symlinks ~/.whatever to ~/.dotfiles/topic/.whatever
 - .vim, .vimrc
 - .profile, .bashrc
 - .ssh (CONFIDENTIAL)
 - .gitconfig
 - .tmux.conf
- install homebrew
- install homebrew cask
- install applications
